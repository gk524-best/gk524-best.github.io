  好了进入正题，今天刷到面试题，一道比较简单的题目，判断一个整数是否是奇数：

public boolean isOdd(int i) {}

  真的很简单，很快我想到答案这样解决：

public boolean isOdd(int i) {
    return i % 2 == 1;
}

  那么传入-1呢，可能优化为：

public boolean isOdd(int i) {
    return i % 2 != 0;
}

  但是这并不是别人想要的结果，最后去看了一下答案：

public boolean isOdd(int i) {
   return (i & 1) == 1;
}

  这个答案确实没有想到，利用位运算，之前没有怎么接触过位运算。在这里记录一下。
